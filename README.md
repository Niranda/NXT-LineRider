# NiraBot - NXT LineRider
Just another line-following nxt robot.

## Important
There is a moving Sensor in the front. It can move in an angle of 180°.
May.. i add some additional information about all wheels (in futur)

## Requirement
* 32bit Java
* LeJOS 0.9.1beta
* NXT
